Day 1

What is the internet?

The internet is a vast network of interconnected computers that communicate using standardized protocols. It allows users to access and share information globally.

HTML

What is HTML?

The first web page was written in HTML (HyperText Markup Language). It is the standard markup language used to create web pages. It structures content on the web, allowing browsers to display text, images, and other elements.

How to use HTML?

HTML uses tags to define elements. Tags are enclosed in angle brackets, like <tag>. Elements can include headings, paragraphs, links, images, and lists. For example:

  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>

HTML documents are structured with a basic layout, including a <head> section for metadata and a <body> section for content.

The first piece of the assignment is to copy and paste song lyrics and format them with HTML. Please follow these steps:

  1. Choose a song and find its lyrics online.
  2. Copy the lyrics into your HTML document.
  3. Use headings for the song title and any section titles.
  4. Format verses as paragraphs.
  5. Use lists for any repeated sections, like the chorus.

HTML Elements to Consider

As you format your lyrics, think about the following HTML elements:

  • <h1> for the main title
  • <h2> for section headings
  • <p> for verses
  • <ul> and <li> for lists
